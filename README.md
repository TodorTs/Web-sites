🍕 Todor's Pizza Website
This is a modern and responsive pizzeria website project. It serves as a showcase of web development skills, featuring a clean design, an interactive menu, and a user-friendly interface.

🚀 Features
Interactive Menu: Browse through a variety of pizzas with descriptions and prices.

Responsive Design: Optimized for all devices, including desktops, tablets, and smartphones.

About Us Section: Information about the pizzeria's history and quality standards.

Contact & Location: Easy-to-find contact details and location information.

Modern UI/UX: Clean aesthetics with smooth navigation.

🛠️ Technologies Used
The project is built using:

HTML5 – for content structure.

CSS3 – for styling, layout (Flexbox/Grid), and animations.

JavaScript – for interactive elements and logic.

📦 How to Run the Project Locally
To view the website on your local machine, follow these steps:

Clone the repository:

Bash
git clone https://github.com/TodorTs/Web-sites.git
Navigate to the project folder:

Bash
cd "Web-site pizzaria"
Launch the site:
Simply open the index.html file in any modern web browser.
